ServiceNow Vulnerability Analyzer API

Azure Functions API for vulnerability analysis and software inventory management

üöÄ Quick Start

Base URL: http://localhost:7071/api (Local) or https://your-function-app.azurewebsites.net/api (Azure)

Authentication: Include API key in header: X-API-Key: your-api-key

Content-Type: All requests must use application/json

üìù Example Requests

Analyze a CVE:

curl -X POST http://localhost:7071/api/analyze-vulnerability \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "vuln_id": "CVE-2024-1234",
    "fetch_all_details": false,
    "confirmation_state": "confirmed"
  }'

Search Software by Manufacturer:

curl -X POST http://localhost:7071/api/software-inventory \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "search_type": "manufacturer",
    "manufacturer": "Palo Alto Networks",
    "fetch_details": true
  }'

Batch Analyze Multiple Vulnerabilities:

curl -X POST http://localhost:7071/api/batch-analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "vuln_ids": ["CVE-2024-1234", "CVE-2024-5678", "QID-110504"],
    "parallel": true
  }'
